类型 函数名 作用 输入参数 返回值 例子
for range 和len结合使用相当于取某范围List下标,第三个参数为步长 N/A N/A for i in range(0, len(nums))
for enumerate 枚举List的下标和数值，与range不同的是它不能指定范围和步长 N/A N/A for i, n in enumerate(nums)
Math max 取最大值 Num Num max_len = max(2, 3)
String str 任意类型变成字符串 Any String str(2)
String replace 替换 String String ‘abc’.replace(‘c’, ‘d’)
String format 格式化字符串 Any String ‘{} of {}’.format(a, b)
String [] 得到某一个字符 N/A Char s[2]
String len 得到字符串长度 N/A int num = len(s)
List len 大小 N/A int num = len(list)
List append 尾部加入 Any N/A list.append(‘apple’)
List join 加入分隔符 List String ‘,’.join(list)
List sort 排序，如果List的元素是tuple，按第一个元素排序 List N/A list.sort()
Dictionary items Dict的所有pairs N/A K,V for k, v in dict.items()
Dictionary items Dict的所有keys N/A List for k in dict.keys()
Dictionary in 是否含有某个key N/A boolean if key in dict
Set N/A 产生一个Set N/A Set b = set()
Lambda func/expr…for…in 整型数组变字符串数组 List List str(x) for x in list
