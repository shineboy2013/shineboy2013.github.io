<pre>Input: p = [1,2,3], q = [1,2,3] Output: true </pre>

Example 2:

<pre>Input: p = [1,2], q = [1,null,2] Output: false </pre>

Example 3:

<pre>Input: p = [1,2,1], q = [1,1,2] Output: false </pre>

Constraints:

  • The number of nodes in both trees is in the range [0, 100].
